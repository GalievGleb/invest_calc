# Roadmap разработки инвестиционного калькулятора

## Описание проекта

Веб-сайт для расчета инвестиций с двумя основными функциями:

1. Расчет времени, необходимого для достижения целевой суммы
2. Расчет ежемесячных отчислений для достижения целевой суммы

**Требования:**

- Темная тема интерфейса
- Без авторизации
- Учет процентной ставки и начального капитала
- Визуализация роста капитала на графике

## Технические детали реализации

### Математические формулы

**Расчет будущей стоимости:**

```
FV = PV × (1 + r)^n + PMT × [((1 + r)^n - 1) / r]
```

где:

- FV - будущая стоимость
- PV - начальный капитал
- PMT - ежемесячное пополнение
- r - месячная процентная ставка (годовая / 12)
- n - количество периодов (месяцев)

**Обратный расчет времени:**
Итеративный метод или использование логарифмов для определения n

**Обратный расчет отчислений:**
Вычисление PMT из формулы будущей стоимости
